## Mixins

To set the required font size, there is a font mixins that are to be used which will set the correct font size to the line height. Import the sass mixin in the scss file to use.

<br />

#### Mobile and Desktop sizing

```html
@include abstracts.get-font-size('xsmall');
```

#### Mobile sizing only

```html
@include abstracts.get-font-size-mobile('xsmall');
```

#### Desktop sizing only

```html
@include abstracts.get-font-size-desktop('xsmall');
```
