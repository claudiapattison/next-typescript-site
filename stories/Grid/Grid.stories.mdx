## Media Queries

While we use rems in most of our projects, pxs are used for grid breakpoints. This is because the viewport width is in pixel and does not change with the font size.

Default Grid breakpoints

- xs - < 576px
- sm - ≥ 576px
- md - ≥ 768px
- lg - ≥ 992px
- xl - ≥ 1200px
